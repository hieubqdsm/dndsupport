## Trang 1 — Thông số cơ bản & Chiến đấu

### Header (Thông tin chung)
| Ô | Nội dung |
|---|---|
| Tên Nhân Vật | Tên nhân vật D&D |
| Lớp Nhân Vật & Level | Class + Level (VD: Fighter 5) |
| Nguồn Gốc | Background |
| Tên Người Chơi | Tên người chơi thật |
| Chủng Tộc | Race (Human, Elf, ...) |
| Phẩm Chất | Alignment (Lawful Good, ...) |
| Điểm Kinh Nghiệm | XP |

### Cột trái — 6 Ability Scores
- **Sức Mạnh** (Strength)
- **Nhanh Nhẹn** (Dexterity)
- **Thể Lực** (Constitution)
- **Sự Thông Minh** (Intelligence)
- **Trí Khôn** (Wisdom)
- **Giao Tiếp** (Charisma)

Mỗi ô gồm: **điểm chính** (score) + **modifier** (số nhỏ bên trên)

### Cột giữa phải — Chiến đấu
- **Lớp Giáp** (AC)
- **Vị Trí Lượt** (Initiative)
- **Di Chuyển** (Speed)
- **Số Máu Tối Đa** (Max HP)

- **Số Máu Hỗ Trợ** (Temporary HP)
- **Xúc Xắc Hồi Phục** (Hit Dice) — Tối Đa + Hiện có
- **Hồi Phục** (Death Saves) — 3 Thành Công ⭕⭕⭕ / 3 Thất Bại ⭕⭕⭕

## Trang 3 — Phép thuật (Spellcasting)

### Header
- **Lớp Nhân Vật Sử Dụng Phép** (Spellcasting Class)
- **Khả Năng Sử Dụng Phép** (Spellcasting Ability)
- **Điểm Tránh Đòn Đánh Từ Phép (DC)** (Spell Save DC)
- **Điểm Hỗ Trợ Tấn Công Cho Phép** (Spell Attack Bonus)

### Bảng phép thuật 10 cấp

| Cấp | Nội dung |
|---|---|
| **0 — Cantrips** | Danh sách phép (không giới hạn slot) |
| **1–9** | Mỗi cấp gồm: Level Phép, Tối Đa Lượt Sử Dụng Phép (Total Slots), Số Phép Đã Sử Dụng (Expended), Tên Phép (danh sách ~13 dòng mỗi cấp) |

Mỗi phép có checkbox **Tên Phép Đã Biết** (⭕) để đánh dấu prepared spells.

| Tên, Class, Level, Background, Race, Alignment, XP | ✅ | Đầy đủ |

| AC, Initiative, Speed | ✅ | |
| HP (current/max/temp) | ✅ | |
| Hit Dice | ✅ | |
| Death Saves | ✅ | |
| Attacks | ✅ | + danh sách 38 vũ khí |

| Other Proficiencies & Languages | ✅ | |
| Equipment | ✅ | |
| Tiền (CP/SP/EP/GP/PP) | ✅ | |
| Features & Traits | ✅ | |
| Ngoại hình (tuổi, cao, nặng, mắt, da, tóc) | ✅ | |
| Backstory | ✅ | |

| Treasure | ✅ | |
