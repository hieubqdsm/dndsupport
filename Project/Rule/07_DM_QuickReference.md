## Hành Động Phổ Biến (Actions)

| Action | Hiệu ứng |
|---|---|
| **Attack** | 1 attack (nhiều hơn nếu Extra Attack) |
| **Cast Spell** | Cast 1 phép (Action) |
| **Dash** | ×2 movement |
| **Disengage** | Tránh Opportunity Attack |
| **Dodge** | Attack vào bạn Disadvantage, DEX save Advantage |
| **Help** | Cho đồng minh Advantage |
| **Hide** | Stealth check → Invisible |
| **Ready** | Chuẩn bị Action với trigger |
| **Search** | Perception / Investigation check |

## Damage Types & Common Resistances

| Type | Damage từ |
|---|---|
| Bludgeoning | Búa, rơi, unarmed |
| Piercing | Giáo, tên, dagger |
| Slashing | Kiếm, rìu |
| Fire | Lửa, Fireball |
| Cold | Băng, Ray of Frost |
| Lightning | Sét, Lightning Bolt |
| Thunder | Sóng âm, Thunderwave |
| Poison | Nọc độc, Poison Spray |
| Acid | Axit, Acid Splash |
| Necrotic | Undead, năng lượng chết |
| Radiant | Thánh quang, Guiding Bolt |
| Force | Magic Missile, Eldritch Blast |
| Psychic | Tấn công tâm linh |

> **B/P/S** = Bludgeoning / Piercing / Slashing. Nhiều quái có Resistance hoặc Immunity với B/P/S damage từ vũ khí không phải magical.

## Conditions — Tóm Tắt Nhanh

| Condition | Hiệu ứng chính |
|---|---|
| **Blinded** | Disadvantage attack, bị attack Advantage |
| **Charmed** | Không tấn công kẻ charm |
| **Deafened** | Không nghe |
| **Exhaustion** | −2 per level mọi d20 roll (10 = chết) |
| **Frightened** | Disadvantage gần nguồn, không lại gần |
| **Grappled** | Speed = 0 |
| **Incapacitated** | Không Action/Bonus/Reaction |
| **Invisible** | Advantage attack, Disadvantage bị attack |
| **Paralyzed** | Incap + fail STR/DEX saves, 5ft hit = crit |
| **Petrified** | Hoá đá, resistance all damage |
| **Poisoned** | Disadvantage attack + ability checks |
| **Prone** | Melee against Adv, Ranged Disadv, ½ speed đứng dậy |
| **Restrained** | Speed 0, Disadv attack, Adv bị attack |
| **Stunned** | Incap + fail STR/DEX, Adv bị attack |
| **Unconscious** | Incap + Prone + crit 5ft |

## Fall Damage
- **1d6** mỗi **10 ft** rơi
- **Max 20d6** (200 ft)
- Land = Prone

---

## Suffocating
- Nín thở: **1 + CON mod** phút (min 30s)
- Sau đó: Drop 0 HP, dying

---

## Random Encounter Pacing

| Situation | Check Frequency |
|---|---|
| Dungeon | Mỗi 30 phút (real dungeon time) |
| Wilderness | Mỗi giờ travel |
| Rest | 1 check per watch (2-3 giờ) |
| Urban | Tùy mood, thường ít check |

## Helpful DM Screen Info

### NPC Attitude
- **Friendly**: Giúp đỡ miễn phí
- **Indifferent**: Hợp tác nếu có lý do
- **Hostile**: Phá hoại, tấn công
