# Monster Manual — Quái Vật Theo CR

> Dữ liệu quái vật D&D 5E (2024) tổ chức theo Challenge Rating.
> Mỗi file chứa 1 nhóm CR, liệt kê đầy đủ stats để DM tra cứu nhanh.

## Cấu Trúc Thư Mục

| File | CR Range | Nội dung |
|---|---|---|
| `CR_0.md` | CR 0 | Động vật, NPC yếu (Commoner, Rat, Cat...) |
| `CR_1_8_to_1_4.md` | CR ⅛ – ¼ | Minions, scouts, low-tier enemies |
| `CR_1_2_to_1.md` | CR ½ – 1 | Thugs, wolves, basic threats |
| `CR_2_to_3.md` | CR 2 – 3 | Ogre, Mimic, Werewolf, Veterans |
| `CR_4_to_5.md` | CR 4 – 5 | Boss nhỏ: Banshee, Troll, Elementals |
| `CR_6_to_8.md` | CR 6 – 8 | Mid-tier bosses: Mage, Young Dragons |
| `CR_9_to_12.md` | CR 9 – 12 | Serious threats: Aboleths, Cloud Giants |
| `CR_13_to_17.md` | CR 13 – 17 | Adult Dragons, Liches, Beholders |
| `CR_18_plus.md` | CR 18+ | Ancient Dragons, Demogorgon, Tarrasque |
