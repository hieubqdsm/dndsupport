### Weapon Properties

| Property | Giải thích |
|---|---|
| **Ammunition** | Cần đạn/tên, nhặt lại ½ sau combat |
| **Finesse** | Chọn STR hoặc DEX cho attack/damage |
| **Heavy** | Small creatures bị Disadvantage |
| **Light** | Dùng Two-Weapon Fighting |
| **Loading** | Chỉ bắn 1 lần/Action (trừ Extra Attack) |
| **Range (X/Y)** | X = tầm thường, Y = tầm xa (Disadvantage) |
| **Reach** | Tầm melee = 10 ft thay vì 5 ft |
| **Thrown (X/Y)** | Ném được, dùng cùng modifier melee |
| **Two-handed** | Cần 2 tay để tấn công |
| **Versatile (XdY)** | 1 tay = damage gốc, 2 tay = damage cao hơn |

### Armor Proficiency
- Mặc giáp **không proficient** → Disadvantage mọi STR/DEX roll + **không cast phép được**
- **Light**: Bard, Ranger, Rogue, Warlock
- **Medium**: Cleric, Druid, Fighter, Paladin, Ranger
- **Heavy**: Cleric (War), Fighter, Paladin

### Tính AC
- **Không giáp**: 10 + DEX mod
- **Light Armor**: Base + DEX mod (full DEX)
- **Medium Armor**: Base + DEX mod (max +2)
- **Heavy Armor**: Fixed (không cộng DEX)
- **Shield**: +2 AC (cộng thêm, 1 tay giữ khiên)

### Khi Level Up:
1. HP tăng: Roll **Hit Die + CON mod** (hoặc lấy trung bình)
2. Proficiency Bonus có thể tăng
3. Nhận **class features** mới (theo bảng class)
4. Lv4, 8, 12, 16, 19: **Ability Score Improvement** (+2 ASI hoặc chọn Feat)
